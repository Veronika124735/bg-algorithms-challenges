# Предизвикателство: Намиране на липсващото число

## Инструкции

Напишете функция `findMissingNumber`, която приема масив от уникални числа от 1 до n (включително), в който едно число липсва. Функцията трябва да върне липсващото число.

### Сигнатура на функцията

```js
/**
 * Returns the missing number in an array of unique numbers from 1 to n (inclusive).
 * @param {number[]} arr - The array of numbers.
 * @returns {number} - The missing number.
 */
function findMissingNumber(arr: number[]): number;
```

### Примери

```js
findMissingNumber([1, 2, 3, 4, 6, 7, 8, 9, 10]); // 5
findMissingNumber([10, 8, 6, 7, 5, 4, 2, 3, 1]); // 9
findMissingNumber([10, 5, 1, 2, 4, 6, 8, 3, 9]); // 7
```

### Ограничения

- Ако бъде подаден празен масив, функцията трябва да върне `1`
- Ако нищо не бъде подадено, функцията трябва да върне `undefined`

### Подсказки

- Изчислете сумата на числата от 1 до n (включително). Формулата за това е `n * (n + 1) / 2`. `n` е дължината на масива плюс 1.
- Изчислете сумата на числата в масива.
- Извадете сумата на числата в масива от сумата на числата от 1 до n (включително).
- Можете да използвате for цикъл или метода reduce за изчисляване на сумата на числата в масива. В следващия раздел ще се фокусираме върху методи като reduce, но все пак ще ви покажа и двата начина.

### Обяснение

Функцията `findMissingNumber` приема масив от уникални числа от 1 до n (включително), в който едно число липсва, и връща липсващото число.

Първо изчисляваме сумата на числата от 1 до n (включително) по формулата `n * (n + 1) / 2`. Това се нарича формула на Гаус. Записваме тази стойност в променлива `expectedSum`.

За по-голяма яснота, нека разгледаме пример. Ако имаме масив от числа от 1 до 10, сумата на тези числа е 55. Можем да я изчислим чрез формулата на Гаус: `10 * (10 + 1) / 2 = 55`. Можем също да я изчислим чрез for цикъл или метода reduce.

След това изчисляваме сумата на числата в масива. Записваме тази стойност в променлива `actualSum`.

Накрая връщаме разликата между очакваната сума и действителната сума. Това ще бъде липсващото число.

Отново, можете да използвате for цикъл или метода reduce за изчисляване на сумата на числата в масива.

### Тестови случаи

```js
test('Finding the missing number', () => {
  expect(findMissingNumber([1, 2, 3, 5])).toBe(4);
  expect(findMissingNumber([10, 8, 6, 7, 5, 4, 2, 3, 1])).toBe(9);
  expect(findMissingNumber([1, 3, 4, 5, 6])).toBe(2);
});
```
