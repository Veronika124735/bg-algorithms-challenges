# Предизвикателство: Намиране на липсващата буква

## Инструкции

Напишете функция `findMissingLetter`, която приема масив от последователни (нарастващи) букви и връща липсващата буква в масива.

### Сигнатура на функцията

```js
/**
 * Returns the missing letter in an array of consecutive letters.
 * @param {string[]} arr - An array of consecutive letters.
 * @returns {string} - The missing letter.
 */
function findMissingLetter(arr: string[]): string;
```

### Примери

```js
findMissingLetter(['a', 'b', 'c', 'd', 'f']); // => "e"
findMissingLetter(['O', 'Q', 'R', 'S']); // => "P"
findMissingLetter(['t', 'u', 'v', 'w', 'x', 'z']); // => "y"
```

### Ограничения

- Входният масив ще съдържа само букви в един регистър (малки или главни)
- Ако масивът не съдържа букви, върнете празен низ

### Подсказки

- Можете да запишете азбуката в низ и да използвате метода `indexOf`, за да получите индекса на буква в низа с азбуката.
- Друг подход е да използвате метода `charCodeAt`, за да получите Unicode стойността на буква.

```js
test('Find Missing Letter', () => {
  expect(findMissingLetter(['a', 'b', 'c', 'e'])).toBe('d');
  expect(findMissingLetter(['X', 'Z'])).toBe('Y');

  expect(findMissingLetter(['F', 'G', 'H', 'J'])).toBe('I');
```
