# Предизвикателство: Валидни IPv4 адреси

## Инструкции

Напишете функция наречена `isValidIPv4`, която приема низ като вход и връща `true`, ако входът е валиден IPv4 адрес в десетичен формат с точки, и `false` в противен случай. IPv4 адресът трябва да се състои от четири октета със стойности между 0 и 255, включително.

### Сигнатура на функцията

```js
/**
 * Checks if a given string is a valid IPv4 address.
 * @param {string} input - The input string to check.
 * @returns {boolean} - True if the input is a valid IPv4 address, false otherwise.
 */
function isValidIPv4(input: string): boolean;
```

### Примери

```js
isValidIPv4('1.2.3.4'); // true
isValidIPv4('123.45.67.89'); // true
isValidIPv4('1.2.3'); // false
isValidIPv4('1.2.3.4.5'); // false
isValidIPv4('123.456.78.90'); // false
isValidIPv4('123.045.067.089'); // false
```

### Ограничения

- Входът ще бъде единичен низ.

### Подсказки

- Можете да използвате метода `split()`, за да разделите низа на части по символа точка.
- Можете да използвате `every()`, за да проверите дали всички октети са в допустимия обхват от 0 до 255.

## Обяснение

- Разделете входния низ на масив от низове, като използвате метода `split()`.
- Проверете дали масивът има точно четири елемента. Ако не — върнете `false`.
- Използвайте метода `every()`, за да проверите дали всички октети са валидни.
  - Преобразувайте октета в число с `parseInt()`.
  - Проверете дали числото е между 0 и 255, включително.
  - Проверете дали октетът е равен на числото, преобразувано обратно в низ. Това е за проверка на водещи нули.
