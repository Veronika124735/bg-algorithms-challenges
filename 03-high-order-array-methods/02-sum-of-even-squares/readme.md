# Предизвикателство: Сума на квадратите на четните числа

## Инструкции

Нека започнем с доста просто предизвикателство, но такова, което използва няколко различни метода за масиви. Напишете функция наречена `sumOfEvenSquares`, която приема масив от числа и връща сумата на квадратите на четните числа в масива.

### Сигнатура на функцията

```js
/**
 * Returns the sum of the squares of the even numbers in the array.
 * @param {number[]} numbers - The array of numbers.
 * @returns {number} - The sum of the squares of even numbers.
 */
function sumOfEvenSquares(numbers: number[]): number
```

### Примери

```js
sumOfEvenSquares([1, 2, 3, 4, 5]); // 20 (2^2 + 4^2)
sumOfEvenSquares([-1, 0, 1, 2, 3, 4]); // 20 (0^2 + 2^2 + 4^2)
sumOfEvenSquares([]); // 0
```

### Подсказки

- Използвайте метода `filter`, за да изберете четните числа от масива.
- След това използвайте метода `map`, за да повдигнете на квадрат всяко четно число.
- Накрая използвайте метода `reduce`, за да сумирате повдигнатите на квадрат четни числа.

function sumOfEvenSquares(numbers) {

  return evenSquares.reduce((sum, square) => sum + square, 0);
}

### Обяснение

Можете да форматирате това по много различни начини. Това е само един пример.

- Създайте променлива наречена `evenSquares` и я задайте равна на резултата от извикването на метода `filter` върху масива `numbers` и след това извикването на метода `map` върху резултата от това.

- За метода `filter` подайте callback функция, която приема число и връща `true`, ако числото е четно, и `false` в противен случай. Използвахме оператора за остатък (`%`), за да проверим дали числото е четно.

- За метода `map` подайте callback функция, която приема число и връща квадрата на това число. Сега имаме масив от квадратите на четните числа.

- Накрая извикайте метода `reduce` върху масива `evenSquares` и подайте callback функция, която приема сума и квадрат и връща сумата плюс квадрата. Задаваме началната стойност на сумата на `0`.

Можем дори да съкратим това, като свържем методите заедно и използваме стрелкова функция:

```js
const sumOfEvenSquares = (numbers) =>
  numbers
    .filter((num) => num % 2 === 0)
    .map((num) => num ** 2)
    .reduce((sum, square) => sum + square, 0);
```

### Тестови случаи

```js
test('Sum of even squares', () => {
  expect(sumOfEvenSquares([1, 2, 3, 4, 5])).toBe(20);
  expect(sumOfEvenSquares([-1, 0, 1, 2, 3, 4])).toBe(20);
  expect(sumOfEvenSquares([])).toBe(0);
});
```
