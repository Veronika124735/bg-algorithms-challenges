# Предизвикателство: Дума с най-висок резултат

## Инструкции

Даден е низ от думи, трябва да намерите думата с най-висок резултат. Всяка буква от думата получава точки според позицията си в азбуката: `a` = 1, `b` = 2, `c` = 3 и т.н.

Трябва да върнете думата с най-висок резултат като низ.

Ако две думи имат еднакъв резултат, върнете думата, която се появява най-рано в оригиналния низ.

Всички букви ще бъдат малки и всички входни данни ще бъдат валидни.

### Сигнатура на функцията

```js
/**
 * Returns the highest scoring word from a string.
 * @param {string} str - The input string.
 * @returns {string} - The highest scoring word.
 */
function highestScoringWord(str: string): string;
```

### Примери

```js
highestScoringWord('man i need a taxi up to ubud'); // 'taxi'
highestScoringWord('what time are we climbing up the volcano'); // 'volcano'
highestScoringWord('take me to semynak'); // 'semynak'
```

### Ограничения

- Входният низ ще съдържа само малки букви и интервали. Не може да включва числа, специални символи или пунктуация.

### Подсказки

- Можете да използвате метода `split`, за да разделите думите.
- Можете да използвате метода `map`, за да изчислите резултата за всяка дума.
- Можете да използвате метода `reduce`, за да намерите думата с най-висок резултат.

- Върнете думата с най-висок резултат, използвайки `highestIndex`.
